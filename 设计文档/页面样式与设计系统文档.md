# 心域奇旅 · 页面样式与设计系统文档

> 本文档基于当前前端实现（Less + Tailwind）整理，与《视觉设计202603123》及各模块设计文档保持一致，作为开发与扩展的样式规范参考。  
> 文档版本：v1.1 · 定稿日期：2026-03-17（v1.1 增补：暖黄棕背景适配、人物头像显示规范）

## 1. 设计目标与风格定位

### 1.1 目标

- 全站统一：**浪漫、温馨、时尚** 的日系动画风相亲互动游戏。
- 视觉关键词：**柔光、渐变、半透明毛玻璃、圆角、精致线条、温暖色调（粉、紫、酒红）**。
- 参考：《恋与制作人》《光与夜之恋》—— 2D 立绘、柔和色彩、时尚服装、丰富表情。

### 1.2 实现方式

- **全局**：Tailwind 做工具类；`index.less` 做基础字体与根布局。
- **页面级**：各页面目录下同名 `.less` 文件（如 `HomePage.less`、`GamePage.less`），采用类名 + 部分 Tailwind，与本文档中的视觉规范一一对应。

### 3.3 字间距（letter-spacing）

- 主标题：0.18em ~ 0.35em  
- 模块标题/按钮：0.2em  
- 副标题：0.15em  
- 正文：0.05em  
- 小标签：0.05em ~ 0.1em  

### 3.4 标题装饰（首页）

- 主标题：`linear-gradient(165deg, #fffef9 → #fce8ec → #f5d8e0 → #f0e6e8)` + `-webkit-text-stroke: 1.5px rgba(255,255,255,0.7)` + `paint-order: stroke fill` + `drop-shadow`。
- 副标题：类似渐变 + 0.75px 描边 `rgba(255,255,255,0.75)`。
- 下划线：140px × 2px，`linear-gradient(90deg, transparent, rgba(255,255,255,0.5), transparent)`，圆角 2px。

### 3.5 暖黄棕背景下的文字可读性

使用 **暖黄棕背景**（如 `/background/bg-01.png`）的页面（首页、创建角色、选择相亲对象）须加强文字与背景的对比，避免发虚、难以辨认：

- **主标题 / 模块标题**：在原有阴影基础上，增加一层略深阴影，例如 `0 1px 2px rgba(0,0,0,0.35)`、`0 2px 6px rgba(0,0,0,0.25)`、`0 0 1px rgba(0,0,0,0.2)`；首页艺术字主标题描边可略加强（如 0.65→0.7），并增加 `0 1px 3px`、`0 2px 8px` 的 drop-shadow。
- **副标题**：颜色可略提亮（如 0.8→0.9），并加 `0 1px 3px rgba(0,0,0,0.25)` 或 `0 2px 4px rgba(0,0,0,0.15)`。
- **表单标签、玩家名称等**：`text-shadow: 0 1px 2px rgba(0,0,0,0.2~0.25)`，保证在暖色背景或毛玻璃条带上清晰可读。

## 6. 毛玻璃（Glassmorphism）

- **背景**：`background: rgba(255,255,255,0.06~0.12)` 或 `rgba(45,35,55,0.5)`、`rgba(55,40,60,0.45)`（深色剧情区）。  
- **模糊**：`backdrop-filter: blur(10px~12px); -webkit-backdrop-filter: blur(12px)`。  
- **搭配**：细边框 + 内高光 `inset 0 1px 0 rgba(255,255,255,0.15~0.25)` + 轻阴影。  
- 用于：顶部状态栏、表单卡片、弹窗、选项区、剧本卡片、档案卡片等。

## 7. 组件规范

### 7.1 主按钮（Primary Button）

- 字体：Noto Serif SC，1rem ~ 1.15rem，600，字间距 0.2em。  
- 内边距：0.85rem ~ 1rem × 2rem ~ 2.75rem。  
- 圆角：9999px。  
- 背景：`linear-gradient(145deg, 主色, 主色深)`；边框 `1px solid rgba(255,255,255,0.4)`。  
- 阴影：见 5.1 主按钮。  
- 悬停：渐变改为 hover 色、`translateY(-2px)`、阴影增强。  
- 点击：可选 `scale(0.98)` 或 `translateY(0)`。

### 7.2 次级按钮（Secondary Button）

- **通用（深色或中性背景）**：背景 `rgba(255,255,255,0.08~0.1)`，毛玻璃 `blur(12px)`；边框 `1px solid rgba(255,255,255,0.45)`；文字 white-98；悬停背景 0.14~0.18，边框 0.6，阴影略增强，`translateY(-2px)`。
- **暖黄棕背景下的次级按钮**（首页、创建角色、选择相亲对象）：为提升与暖色背景的区分度，采用略加深的底与更清晰边框，避免“浮白”感。  
  - 背景：`rgba(50,40,35,0.25)` 或 `rgba(55,45,40,0.22)`（暖灰棕半透明），保留 `backdrop-filter: blur(12px)`。  
  - 边框：`1px solid rgba(255,255,255,0.5)`，悬停 `0.6`。  
  - 阴影：`0 2px 12px rgba(0,0,0,0.15), inset 0 1px 0 rgba(255,255,255,0.1)`；悬停 `0 4px 16px rgba(0,0,0,0.2), inset 0 1px 0 rgba(255,255,255,0.15)`。  
  - 文字：white-98 + `text-shadow: 0 1px 2px rgba(0,0,0,0.2)`。  
  - 悬停：背景加深至约 `rgba(60,50,45,0.35)`，`translateY(-2px)`。

### 7.3 人物头像 / 立绘显示（避免头部被裁切）

半身立绘、候选人头像等以「头部在上」的竖图为主，须保证脸部、头顶不被裁切：

- **容器**：固定高度时建议不少于 180px～200px（如约会对象选择页候选人卡片头像区 200px），并设 `min-height`；容器使用 `display: flex; align-items: flex-start; justify-content: center` 使内容从顶部对齐。
- **图片**：`object-fit: cover` 时**必须**配合 **`object-position: top center`**，使裁切以顶部为基准，优先保留头部；若为首页等 3:4 区域，同样建议 `object-position: top`（或 Tailwind `object-top`）。
- **卡片高度**：若头像区增高，同步提高卡片与卡片内层的 `min-height`（如 300px→380px），避免布局挤压或遮挡头像区域。

适用页面：选择相亲对象（候选人卡片）、首页人格卡片（已用 `object-top`）、创建角色 MBTI 卡片等。

### 7.4 卡片（通用）

- 圆角：16px ~ 20px。  
- 背景：毛玻璃或深色半透明（如 `rgba(12,12,24,0.78~0.85)`）。  
- 边框：1px 白色低透明度或深色描边。  
- 悬停：`scale(1.03~1.05)`、`translateY(-4px~-6px)`、阴影加深。  
- 过渡：`transform`、`box-shadow` 约 0.2s~0.25s ease。

### 7.5 表单控件

- **输入框**：圆角 12px，背景 `rgba(255,255,255,0.08)`，边框 white-45，padding 0.75rem 1rem，placeholder white-45。  
- **性别/选项按钮**：胶囊或圆角 12px，未选 0.06 背景，选中主色或主色半透明（0.35）。  
- **滑块**：轨道 8px 高、圆角 4px、背景 white-2；滑块 18px~20px 圆、主色、轻阴影。  
- **标签**：Noto Serif SC 或常规 1rem，600，white-98，字间距 0.12em。

### 7.6 弹窗 / Modal

- 遮罩：`rgba(0,0,0,0.4~0.5)` + `backdrop-filter: blur(4px)`。  
- 内容区：max-width 480px，圆角 1.25rem，毛玻璃背景，边框 white-45，阴影 0 24px 48px。  
- 标题栏：下边框 1px white-15，关闭按钮圆形、white-08 背景、hover 略放大。  
- 入场/离场：opacity + scale(0.92) + translateY(12px)，时长约 0.28s。

### 7.7 MBTI 标签

- 小标签：padding 0.2rem 0.5rem ~ 0.25rem 0.6rem，圆角 6px~8px，字号 0.75rem~0.85rem，字重 600，白色字，背景为对应 MBTI 色。

### 7.8 进度条（好感度）

- 容器：宽 80px，高 8px，圆角 4px，背景 `rgba(0,0,0,0.25)`。  
- 填充：`linear-gradient(90deg, 主色, 主色深)`，`transition: width 0.3s ease`。  
- 可配爱心图标（#e84a6f）+ 数值。

## 8. 动效与过渡

### 8.1 时长

- 按钮：0.22s。  
- 卡片：0.25s。  
- 弹窗：0.28s。  
- 进度条：0.3s。

### 8.2 常用效果

- 按钮/卡片悬停：`translateY(-2px ~ -6px)`，部分 `scale(1.03~1.05)`。  
- 按钮点击：`scale(0.98)` 或 `translateY(0)`。  
- 弹窗：opacity + scale(0.92) + translateY(12px)。  
- 爱心泡泡：自下而上线性位移 + 轻微横向漂移，时长 14s~21s 随机，延迟错开。  
- 成功提示：`translateY(-4px)` + opacity，0.25s ease。

## 9. 各页面样式摘要

### 9.1 首页（HomePage）

- 背景：`/background/bg-01.png` 全屏 cover，min-width 1280px（**暖黄棕背景**，须采用 3.5、7.2 暖黄棕适配）。  
- 装饰：爱心泡泡雨（♥），透明度与字号随机，自底向上动画。  
- 标题：艺术字主副标题 + 渐变描边（主标题描边 0.7、多层 drop-shadow）+ 下划线。  
- 中部：4 列网格卡片，min 280×360，背景 `card.png` + MBTI 色 overlay 20%~25%，头像 3:4 圆角、**object-position: top** 保证头部完整，hover 卡片 scale 1.05、上移、阴影加深。  
- 底部：三按钮（玩家档案、玩法介绍、开始游戏），主按钮酒红渐变；次级按钮使用暖黄棕背景变体（暖灰棕半透明底、白边、文字阴影）。  
- 玩法介绍：弹窗毛玻璃、圆角 1.25rem、自定义滚动条（6px、white-06/25）。

### 9.2 剧本大厅（ScriptHall）

- 背景：全屏背景图 + 毛玻璃内容区。  
- 标题 40px、字间距 2px、阴影 0 4px 16px。  
- 剧本卡片：3 列网格，圆角 18px，深色半透明 `rgba(12,12,24,0.85)`，卡片背景图 opacity 0.18，hover 上移 6px、阴影加深；锁定态 opacity 0.6。  
- 按钮：胶囊形，次级为深色半透明 + 白边。

### 9.3 创建角色（CreateRole）

- 背景：`bg-01.png`（**暖黄棕背景**），与首页同主色、毛玻璃、Noto Serif SC。  
- 标题/副标题：采用 3.5 节暖黄棕文字可读性规范（多层 text-shadow）；表单标签带 `0 1px 2px rgba(0,0,0,0.2)`。  
- 表单卡片：max-width 560px，圆角 1.25rem，毛玻璃 + 内高光。  
- 性别：两枚胶囊按钮，选中主色填充。  
- 年龄：滑块，滑块头主色 20px 圆。  
- 职业：5 列网格，圆角 12px，选中主色 0.35 背景。  
- MBTI：4 列卡片，3:4 头像（建议 object-position: top）+ 标签 + 一句话，选中白 0.12 背景。  
- 一句话介绍：输入框 + 字数 + 可选「随机生成」按钮。  
- 底部：取消/确认，主按钮与首页一致；次级按钮使用暖黄棕背景变体。

### 9.4 选择相亲对象（BlindDatePage）

- 背景：`bg-01.png`（**暖黄棕背景**），与创建角色、首页风格一致。  
- 标题/副标题、玩家名称：采用 3.5 节暖黄棕文字可读性规范；玩家名称单独设字重与 text-shadow。  
- 顶部：当前玩家 MBTI + 一句话，毛玻璃胶囊条。  
- 卡片：5 张横排，220px 宽、**min-height 380px**，毛玻璃内层、圆角 1.25rem；**头像区 200px 高**，`object-position: top center` 保证头部不被裁切（见 7.3）；信息区姓名/年龄/职业/MBTI/一句话，hover scale 1.03。  
- 底部：返回修改角色、换一批、随机匹配（次级使用暖黄棕变体），确认为主色。  
- 确认弹窗：深色半透明 `rgba(30,25,35,0.95)`，毛玻璃，圆角 1.25rem。

### 9.5 游戏剧情（GamePage）

- 背景：章节背景图全屏。  
- 顶部状态栏：毛玻璃条，圆角 16px，章节名 + 双方头像 40px 圆 + MBTI 标签 + 好感度（爱心 + 80px 进度条 + 数值）。

## 12. 维护说明

- 新增页面时：在页面目录下新增同名 `.less`，复用本文档中的色彩变量（可抽成公共 variables.less 再按需引用）、圆角、阴影、毛玻璃与按钮/卡片规范。  
- **使用暖黄棕背景（如 bg-01.png）时**：须采用第 3.5 节「暖黄棕背景下的文字可读性」与第 7.2 节「暖黄棕背景下的次级按钮」规范，保证字体与按钮在暖色背景上清晰可辨。  
- **人物头像/半身立绘**：凡以竖图、头部在上的展示，须遵循第 7.3 节（object-position: top、容器高度与 min-height），避免头部被裁切。  
- 修改主色或字体时：同步更新本文档与《视觉设计》文档，并在各页面 Less 中统一替换变量。  
- 移动端适配：在现有 PC 基准上增加断点与响应式字号/间距，保持毛玻璃、主色与圆角规范不变。
