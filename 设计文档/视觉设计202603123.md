# 游戏视觉设计文档：MBTI 相亲互动游戏（精致日系动画风格）

---

## 1. 设计目标与风格参考
### 1.1 目标
打造一款画风精致、色彩柔美、角色生动的日系动画风格相亲互动游戏。视觉上要传达浪漫、温馨、时尚的氛围，让玩家沉浸于与 MBTI 性格角色的互动中。

### 1.2 风格参考
- **核心参考**：《恋与制作人》《光与夜之恋》——细腻的 2D 立绘、柔和的色彩过渡、时尚的服装设计、丰富的表情、浪漫的场景。
- **视觉关键词**：柔光、渐变、半透明毛玻璃、圆角、精致线条、温暖色调（粉、紫、橙、蓝）、梦幻光效。

## 2. 角色视觉设计
### 2.1 玩家角色（不显示立绘）
玩家通过以下元素在界面中代表自己：
- **姓名**：自定义，显示在顶部状态栏。
- **MBTI 标签**：四种人格对应专属配色与符号图标。
  | MBTI | 代表色 | 图标符号 | 备注 |
  |------|--------|----------|------|
  | ENFJ | 暖橙色 (#FF9F7C) | 太阳 | 热情、温暖 |
  | INTJ | 冷紫色 (#7C6A9F) | 几何棱锥 | 理性、锐利 |
  | INFJ | 森林绿 (#6B8E7C) | 羽毛 | 温柔、神秘 |
  | ENTJ | 酒红色 (#B84A62) | 王冠 | 自信、领导力 |
- **职业图标**：5 种职业（医生、教师、工程师、设计师、律师）均设计为圆形背景的简约图标，尺寸 60x60px，与 MBTI 图标风格统一。

### 2.2 相亲对象角色
共 5 位候选人，每位需具备独立立绘、姓名、年龄、职业、MBTI、一句话介绍。

#### 2.2.1 角色视觉特征（基于 MBTI 与职业）
- **ENFJ（主人公型）**  
  - 外形：温暖阳光，笑容灿烂，眼神亲切。  
  - 服饰：柔和色调（米白、浅粉、淡蓝），休闲时尚（针织衫、连衣裙），配饰简约。  
  - 职业细节：医生（听诊器挂脖）、教师（手持书本）、工程师（平板电脑）、设计师（画笔）、律师（文件夹）——以小配饰点缀。

- **INTJ（建筑师型）**  
  - 外形：冷静知性，眼神深邃锐利，姿势内敛。  
  - 服饰：简约深色（黑、深蓝、灰），剪裁利落（衬衫、西装、高领），金属质感配饰。  
  - 职业细节：医生（白大褂内搭）、教师（眼镜+教案）、工程师（机械笔）、设计师（素描本）、律师（钢笔）。

- **INFJ（提倡者型）**  
  - 外形：温柔神秘，若有所思，略带书卷气。  
  - 服饰：柔和中性色（灰绿、淡紫、奶茶色），材质轻柔（棉麻、毛衣），可能佩戴细边眼镜。  
  - 职业细节：医生（柔和色调的听诊器）、教师（旧书）、工程师（创意草图）、设计师（水彩盘）、律师（卷宗）。

- **ENTJ（指挥官型）**  
  - 外形：自信强势，姿态挺拔，目光坚定。  
  - 服饰：干练优雅（西装套装、高定裙装），深色或饱和色（酒红、深蓝），配饰精致（手表、胸针）。  
  - 职业细节：医生（高级听诊器）、教师（教鞭）、工程师（蓝图）、设计师（设计稿）、律师（法槌模型）。

#### 2.2.2 立绘规格
- **类型**：半身立绘（胸部以上），带手部动作（可手持职业配饰）。
- **尺寸**：800x1000px，透明背景 PNG。
- **表情差分**：每位角色需 3~4 种表情（普通、微笑、惊讶、思考/严肃），确保脸型、发型、服装一致。
  - 文件名规则：`candidate_01_base.png`，`candidate_01_smile.png`，`candidate_01_surprise.png`，`candidate_01_think.png`（01~05 代表 5 位角色）。

### 2.3 角色生成指南（AIGC）
使用 **Midjourney** 生成基础立绘，再利用 **Photoshop** 或 **移除背景工具** 处理为透明底。

#### 提示词模板（以 ENFJ 女性为例）
```
A beautiful young woman in her late 20s, warm smile, soft pink blouse, stylish modern outfit, gentle eyes, ENFJ personality, anime style, detailed face, soft lighting, pastel colors, clean background, high quality, 8k --ar 2:3 --niji 5
```
- 关键点：描述性格特征、服装、表情、风格。
- 使用 `--seed` 固定种子，便于生成同一角色的不同表情。

#### 表情差分生成
在基础描述中替换表情词：
- 微笑：`smiling gently`
- 惊讶：`surprised expression, eyes wide`
- 思考：`thoughtful look, looking away slightly`

#### 一致性保障
- 使用 Midjourney 的 `--seed` 参数（如 `--seed 12345`）保证构图相似。
- 或使用 Stable Diffusion + ControlNet（Canny 或 Depth）固定姿势。

## 3. 场景视觉设计
### 3.1 四章场景（各 1 张背景）
| 章节 | 场景描述 | 氛围 | 色调 |
|------|----------|------|------|
| 第一章（开头） | 咖啡厅内景：午后阳光，木质桌椅，绿植，窗外街景 | 温馨、轻松 | 暖黄、米白 |
| 第二章（发展） | 公园/河边步道：樱花或绿树，长椅，小桥流水 | 清新、浪漫 | 粉绿、淡蓝 |
| 第三章（高潮） | 高级法式餐厅：烛光晚餐，水晶吊灯，夜景 | 优雅、暧昧 | 深紫、暖橙 |
| 第四章（结局） | 海边日落/城市天台：晚霞，海面，或霓虹灯光（根据结局可微调） | 感动/遗憾 | 晚霞色/灰蓝 |

- 背景尺寸：1920x1080px，JPG 或高质量 PNG。
- 文件名：`bg_chapter1.jpg`，`bg_chapter2.jpg`...

### 3.2 其他界面背景
- **角色选择界面**：简约的模糊渐变背景（可复用咖啡厅背景的高斯模糊版）。
- **结局界面**：根据结局氛围，可用同一海边背景叠加不同滤镜（暖色/冷色）。

### 3.3 生成指南
使用 Midjourney 生成，提示词示例：
```
cozy coffee shop interior, afternoon sunlight, wooden tables, plants, people sitting, anime art style, detailed, soft colors, 16:9 --niji 5
```
后期可在 Photoshop 中添加柔光、景深效果。

## 4. UI 界面视觉设计
### 4.1 整体风格
- **主色调**：粉紫渐变 (#FAD0E6 → #D9B0E0)，参考恋与制作人。
- **辅助色**：各 MBTI 代表色用于标签、进度条等。
- **材质**：半透明毛玻璃效果（背景模糊 10px，白色 30% 透明度），圆角，细线边框，柔和阴影。
- **字体**：圆体（如汉仪PP体、方正兰亭圆体_GBK），标题 24px，正文 18px，选项按钮 20px。

### 4.2 主要界面设计
#### 4.2.1 启动界面
- 背景：高质量角色群像或浪漫场景插画。
- Logo：艺术字 + 英文 subtitle，居中。
- 开始按钮：毛玻璃圆角矩形，粉紫色边框，内发光。

#### 4.2.2 角色创建界面
- 表单卡片：白色圆角卡片，内边距 24px。
- 性别选择：两个圆形按钮（男/女），选中时填充粉色。
- 年龄滑条：自定义滑块样式（心形）。
- 职业图标：展示在选项旁，点击选中高亮。
- MBTI 选择：四个卡片，展示代表色和图标，点击选中。
- 一句话介绍：输入框，带字数限制。

#### 4.2.3 选择对象界面
- 5 张候选人卡片（横向滑动或网格）：
  - 左侧小立绘（200x250px 缩略图）
  - 右侧信息：姓名、年龄、职业、MBTI 标签、一句话介绍
  - 卡片悬停轻微放大，点击进入详情确认。
- 底部“随机匹配”按钮（快速选择随机候选人）。

#### 4.2.4 剧情互动界面
- **顶部状态栏**（毛玻璃条）：
  - 左侧：章节名 + 圆点进度（如“第一章·开头 ●○○○”）
  - 中间：双方 MBTI 图标 + 姓名（如“ENFJ 李华” vs “INTJ 林雪”）
  - 右侧：好感度进度条（心形图标 + 填充条，数字可选显示）
- **中部对话区域**：
  - 左侧：对方半身立绘（占 1/3 宽度）
  - 右侧：对话框（圆角矩形，带小尾巴指向立绘），文字逐字显示。
  - 可添加“历史记录”按钮（小图标）。
- **底部选项区域**：
  - 2~3 个圆角按钮，等宽排列，背景毛玻璃，文字黑色。
  - 按钮点击有缩小 + 发光反馈。

#### 4.2.5 结局界面
- 背景：对应结局氛围的插画（可 AIGC 生成）。
- 大标题：结局名称（如“灵魂伴侣”）。
- 评语文本框：毛玻璃卡片。
- 按钮：“重新开始”“分享”（分享暂不可用，可预留）。

### 4.3 UI 组件规格
| 组件 | 尺寸/样式 | 状态 |
|------|-----------|------|
| 按钮 | 圆角 30px，内边距 12px 24px，字体 20px | 常态：白色半透明，粉色边框；悬停：放大 1.05，边框发光 |
| 卡片 | 圆角 20px，白色背景，阴影 0 4px 10px rgba(0,0,0,0.1) | - |
| 进度条 | 高度 10px，宽 200px，背景浅灰 (#E0E0E0)，前景粉紫渐变，心形端点 | - |
| MBTI 图标 | 圆形直径 60px，背景色为对应 MBTI 色，内嵌白色符号 | - |
| 职业图标 | 圆形直径 60px，背景浅灰，内嵌简约图标 | - |
| 对话框 | 最大宽度 500px，圆角 20px，白色半透明，小尾巴 | - |

### 4.4 AIGC 生成 UI 元素示例
- **心形图标**：`flat icon, heart shape, gradient pink, minimalistic, transparent background, vector style`
- **MBTI 符号**：`flat icon, simple sun symbol, orange gradient, minimalistic, transparent background`
- **毛玻璃纹理**：`glass morphism texture, soft blur, pastel gradient, abstract`

## 5. 动画与特效设计
### 5.1 交互动效
- **按钮点击**：缩放至 0.95，恢复，伴随白色闪光。
- **好感变化**：
  - 进度条数值平滑变化，心形跳动。
  - 增减时飘浮数字（+5 / -3），颜色绿/红。
- **选项出现**：底部按钮依次淡入上滑。
- **章节切换**：场景淡入淡出，叠加光斑或花瓣飘落（根据场景）。

### 5.2 剧情表现
- **对话文字**：逐字显示，可点击加速。
- **立绘表情切换**：瞬间切换，或短促淡入（0.2s）。
- **关键选择**：选项按钮高亮，背景轻微变暗。

### 5.3 制作工具
- 简单动效可用 **CSS 动画** 实现。
- 复杂特效可导出为 **Lottie**（用 After Effects + Bodymovin），或使用 **即梦/可灵** 生成视频背景。

## 6. AIGC 制作指南
### 6.1 推荐工具与用途
| 工具 | 用途 | 优势 |
|------|------|------|
| Midjourney | 角色立绘、场景、图标生成 | 风格成熟，画质高 |
| Stable Diffusion | 角色一致性生成，表情差分 | 可控性强（ControlNet） |
| 即梦/可灵 | 动画背景、动态立绘、视频特效 | 国产，易上手 |
| Photoshop | 后期处理，背景去除，调色 | 专业图像处理 |
| Figma | UI 设计，切图 | 协作方便 |

### 6.2 工作流程建议
1. **概念设计**：用 Midjourney 生成多组风格试验图，确定最终风格方向。
2. **角色生成**：
   - 为每位候选人设计基础立绘，保存种子值。
   - 用相同种子 + 修改表情词生成表情差分。
   - 批量导入 Photoshop，用“选择主体”或插件（如 Remove.bg）去背景，调整色调统一。
3. **场景生成**：为四章生成背景，注意透视与角色比例协调。
4. **UI 设计**：在 Figma 中绘制，可参考 Midjourney 生成的 UI 组件灵感图。
5. **资源输出**：按规格导出 PNG/JPG，命名规范。

### 6.3 提示词库
#### 角色立绘（男性模板）
```
A handsome young man in his late 20s, [MBTI traits], wearing [clothing], holding [occupation prop], [expression], anime style, detailed face, soft lighting, [color scheme], clean background, high quality, 8k --ar 2:3 --niji 5
```
- MBTI traits 示例：
  - ENFJ: warm smile, approachable
  - INTJ: cold gaze, sharp features
  - INFJ: thoughtful eyes, gentle aura
  - ENTJ: confident posture, commanding presence

#### 场景背景
```
[location], [time/lighting], anime art style, detailed, soft colors, 16:9 --niji 5
```
例如：`French restaurant interior, candlelight dinner, evening, romantic atmosphere`

#### 图标
```
flat icon, [subject], gradient [color], minimalistic, transparent background, vector style
```

## 8. 总结
本视觉设计文档为 MBTI 相亲互动游戏提供了全面的视觉系统规范，从角色、场景、UI 到动画，并给出了详细的 AIGC 制作指南。遵循此文档，可确保游戏视觉风格统一精致，同时利用 AIGC 工具快速产出高质量资源，支撑 MVP 版本在一周内顺利上线。后续可根据玩家反馈逐步丰富角色表情、场景细节和特效，打造更具沉浸感的互动体验。
