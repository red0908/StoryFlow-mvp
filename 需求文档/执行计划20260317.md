## 三、风险与应对

| 风险 | 应对 |
|------|------|
| 音频自动播放被浏览器拦截 | 所有播放放在用户点击之后；必要时首页/剧本大厅增加「点击开启声音」提示 |
| 部署后刷新非根路径 404 | 使用 `vercel.json` 的 rewrites 将 `/(.*)` 指向 `/index.html` |
| 第二剧本 modern_love_2 无独立节点 | 可暂时与 modern_love 共用一套 nodes，或保留「开发中」占位，MVP 以一套完整剧本为主 |
| 真机样式异常 | 以 Chrome 设备模拟先修；真机只做关键路径验收，细节可记入后续迭代 |
